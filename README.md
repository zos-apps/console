## About

System console for viewing logs and debugging applications.

## Features

- Real-time logs
- Log filtering
- Search
- Export logs

## Installation

### From App Store
Open the **App Store** in zOS and search for "Console".

### Via npm
```bash
npm install @zos-apps/console
```

## Usage

- View system messages
- Filter by app or level
- Search for specific entries



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
